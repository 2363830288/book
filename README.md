# 小书屋
一款网络小说阅读器📕,采用纯flutter编写，支持追书、看书等功能 
## 使用技术 
dart,provider,eventbus,dio等
## 项目已经获得Jetbrain免费license 
感谢Jetbrain 的支持 😀😀😀
